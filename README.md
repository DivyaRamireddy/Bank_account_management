# Bank Account Management System (OOP)

A simple Python program that demonstrates core Object-Oriented Programming concepts through a bank account system.

## Concepts Used
- Encapsulation
- Abstraction
- Inheritance
- Polymorphism

## Features
- Deposit and withdraw money
- Interest calculation for savings account
- Different behavior for savings and current accounts

## How to Run
```bash
python bank_oops.py# Bank_account_management
